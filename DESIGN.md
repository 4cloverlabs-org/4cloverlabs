## Typography Principles

-   **Consistency:** Use a single typeface family across all UI.
-   **Weight Hierarchy:** 400 for body, 500 for interactive text, 700
    for labels.
-   **Spacing Rhythm:** Larger text uses tighter line-height ratios;
    smaller text uses more generous spacing.
-   **Accessibility:** Minimum 14px for actionable links and 16px for
    primary inputs.
-   **Scale Progression:** Larger headings establish hierarchy while
    body text prioritizes readability.
-   **No Decoration:** Avoid underlines, italics, and text shadows. Use
    weight, size, and color for emphasis.
